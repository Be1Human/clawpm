# ClawPM 键盘快捷键文档

> 最后更新：2026-03-11

---

## 全局快捷键

在任意页面均可使用。

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+K` / `Cmd+K` | 打开/关闭命令面板 | 可搜索任务、跳转页面、切换项目 |
| `Escape` | 关闭当前弹窗/面板 | 适用于命令面板、创建任务弹窗、身份切换弹窗 |

---

## 命令面板（`Ctrl+K`）

打开命令面板后可用的快捷键（由 `cmdk` 库提供）。

| 快捷键 | 功能 |
|--------|------|
| 直接输入 | 实时搜索任务（标题/ID） |
| `↑` / `↓` | 在结果列表中导航 |
| `Enter` | 执行选中的命令/跳转 |
| `Escape` | 关闭命令面板 |

**命令面板支持的操作：**

| 类别 | 操作 |
|------|------|
| 任务搜索 | 按标题或任务 ID 搜索，最多显示 10 条 |
| 导航 | 跳转到：项目仪表盘、任务列表、看板、需求树、思维导图、甘特图、迭代管理、归档箱、需求池、里程碑、我的仪表盘、我的任务 |
| 操作 | 创建新任务、切换项目 |

## 思维导图（`/mindmap`）

### 节点选中状态下（全局）

> 当焦点在输入框/文本区域时，以下快捷键不生效。

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 为选中节点添加**子节点** |
| `Enter` | 在选中节点同级添加**兄弟节点** |
| `Delete` / `Backspace` | 删除选中节点（弹出确认对话框） |

### 节点内联编辑

双击节点进入编辑模式后：

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 确认并保存标题修改 |
| `Escape` | 取消编辑，恢复原标题 |

---

## 任务详情（`/tasks/:id`）

| 快捷键 | 场景 | 功能 |
|--------|------|------|
| `Enter` | 内联编辑字段（标题、状态等） | 确认保存 |
| `Escape` | 内联编辑字段 | 取消编辑，还原原值 |
| `Ctrl+Enter` | 任务描述 textarea | 保存描述内容 |
| `Enter` | 备注输入框（有内容时） | 提交备注 |
| `Enter` | 子任务标题输入框（有内容时） | 创建子任务 |

---

## 创建任务弹窗

通过侧边栏「新建」按钮或命令面板「创建新任务」触发。

| 快捷键 | 场景 | 功能 |
|--------|------|------|
| `Enter` | 标题输入框（有标题时） | 提交创建任务 |
| `Enter` | 自定义标签输入框 | 添加当前输入为标签 |
| `Escape` | 任意位置 | 关闭弹窗，放弃创建 |

---

## 侧边栏导航

| 快捷键 | 场景 | 功能 |
|--------|------|------|
| `Enter` | 新建项目输入框 | 确认创建项目 |
| `Escape` | 新建项目输入框 | 取消新建，收起输入框 |

## 迭代详情（`/iterations/:id`）

| 快捷键 | 场景 | 功能 |
|--------|------|------|
| `Enter` | 添加任务输入框（有内容时） | 将任务加入迭代 |
